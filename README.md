# Google Clone - Interfaz Web 

## Descripción
Este proyecto es una **clonación de la interfaz de Google**, desarrollado como ejercicio del **curso de Diseño Web**  
Demuestra habilidades de **maquetación profesional, HTML semántico, CSS moderno (Flexbox/Grid).

---

## Demo
- Abre el proyecto en tu navegador directamente desde GitHub Pages:  
https://magdalenaayude.github.io/google-clone/

--

## Funcionalidades
- Interfaz minimalista inspirada en Google.
- Barra de búsqueda y botones (simulados, sin funcionalidad real de búsqueda).
- Diseño **responsive** para mobile, tablet y desktop.
- Organización de **assets** (imágenes y logos) correctamente referenciados.

---

## Tecnologías
- HTML5 semántico
- CSS moderno (Flexbox / Grid, responsive)
- Carpetas organizadas para assets (imágenes, iconos, logos)
